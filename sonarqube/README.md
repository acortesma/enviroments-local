# Sonarqube

### Introducción
Veremos como disponer en local de Sonarqube


### Instalación

Para iniciar ubícate en el directorio y ejecuta:
```
docker-compose -d up
``` 

Puedes acceder a la pantalla de inicio:
```
http://localhost:9000
```
